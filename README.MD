python3 app.py
```

```
 
## Desactivation du swap 
1. Se connecter a la Debian en sudo 
2. Edition du ficher grub 

```
nano /etc/default/grub 
```

```
GRUB_CMDLINE_LINUX="cgroup_enable=memory swapaccount=1"
```
3. Mettre a jour le grub

```
sudo update-grub 
```
